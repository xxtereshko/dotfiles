# dotfiles README

Clone this repo:

```shell
$ git clone https://github.com/xxtereshko/dotfiles
```

1. Install XCode CLI:

```shell
$ xcode-select --install
```

2. Install Homebrew:

```shell
$ curl -fsSL -o install.sh https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh
```

3. Install Homebrew Packages:

```shell
$ sh brew.sh
```

4. Copy fonts

```shell
$ sh fonts.sh
```

5. Copy config files

```shell
$ sh conf.sh
```
